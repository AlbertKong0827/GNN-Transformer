# C-GMVAE
A PyTorch implementation of CNN-RNN for Crop Yield Prediction

# Dependencies
- Python 3.7+
- PyTorch 1.5.0
- numpy 1.17.3

Older versions might work as well.

# Run
To train the model:
```./run_train.sh```

To test the model, please update the checkpoint `-cp` in `run_test.sh`, and:
```./run_test.sh```
